# Testing Strategy

## Unit Tests

Every module must include unit tests.

Examples

- DOM extraction
- Graph generation
- Workflow extraction

---

## Integration Tests

Test complete browser interactions.

Examples

- Login
- Navigation
- Form filling

---

## End-to-End Tests

Test against a staging website.

Verify

- Workflow generation
- Graph integrity
- Storage

---

## Performance Tests

Measure

- Crawl speed
- Memory
- Graph generation

---

## Regression Tests

Every discovered bug requires a regression test.

---

## Success Criteria

- All tests pass
- No critical errors
- Graph generated successfully
- SQLite populated
- Embeddings created
